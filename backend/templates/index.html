Welcome to AgriFinance API

Quick Start Guide

Use these curl commands to test the API:

1. Register a new user:

curl -X POST https://agrifinance-api.onrender.com/api/auth/register/ \
-H "Content-Type: application/json" \
-d '{
    "username": "testuser",
    "password": "testpass123",
    "email": "test@example.com"
}'

2. Login to get token:

curl -X POST https://agrifinance-api.onrender.com/api/auth/login/ \
-H "Content-Type: application/json" \
-d '{
    "username": "testuser",
    "password": "testpass123"
}'

3. Access protected endpoints:

curl -X GET https://agrifinance-api.onrender.com/api/farmers/ \
-H "Authorization: Bearer your_access_token_here"
