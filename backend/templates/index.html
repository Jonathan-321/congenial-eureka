Access Admin Panel

API Testing Guide

If you need to test the API programmatically, here are some working examples:

1. Login as admin to get tokens:

curl -X POST https://agrifinance-api.onrender.com/api/auth/login/ \
-H "Content-Type: application/json" \
-d '{
    "username": "your_admin_username",
    "password": "your_admin_password"
}'

# Save the access token:
export TOKEN="your_access_token_here"

2. Browse API resources:

# Get API root to discover available endpoints
curl -X GET https://agrifinance-api.onrender.com/api/loans/ \
-H "Authorization: Bearer $TOKEN"

# Get loan products
curl -X GET https://agrifinance-api.onrender.com/api/loans/products/ \
-H "Authorization: Bearer $TOKEN"

# Get loans
curl -X GET https://agrifinance-api.onrender.com/api/loans/loans/ \
-H "Authorization: Bearer $TOKEN"

# Get farmers
curl -X GET https://agrifinance-api.onrender.com/api/farmers/ \
-H "Authorization: Bearer $TOKEN"

3. Create resources (working examples):

# Create loan product
curl -X POST https://agrifinance-api.onrender.com/api/loans/products/ \
-H "Authorization: Bearer $TOKEN" \
-H "Content-Type: application/json" \
-d '{
    "name": "Loan Product Name",
    "description": "Description",
    "min_amount": 100,
    "max_amount": 5000,
    "interest_rate": 12.5,
    "duration_days": 180,
    "repayment_schedule_type": "FIXED",
    "is_active": true
}'

# Create farmer
curl -X POST https://agrifinance-api.onrender.com/api/farmers/ \
-H "Authorization: Bearer $TOKEN" \
-H "Content-Type: application/json" \
-d '{
    "name": "Farmer Name",
    "phone_number": "+250789123456",
    "location": "Location",
    "farm_size": 2.5
}'

# Create loan
curl -X POST https://agrifinance-api.onrender.com/api/loans/loans/ \
-H "Authorization: Bearer $TOKEN" \
-H "Content-Type: application/json" \
-d '{
    "farmer": 5,
    "loan_product": 1,
    "amount_requested": 2000,
    "amount_approved": 2000,
    "status": "PENDING"
}'
Warning: The crop cycle API endpoint is currently experiencing issues. Please use the admin interface to create and manage crop cycles.
